# Однослойный перцептрон 

## linal_lab2

Реализация однослойного перцептрона **с нуля на NumPy** (без PyTorch, TensorFlow, sklearn.linear_model).

## Структура проекта

```
├── perceptron.py    — класс Perceptron (sigmoid, forward, loss, fit, predict)
├── data.py          — подготовка данных (генерация, train/test split, стандартизация)
├── plots.py         — вспомогательные функции визуализации
├── train.py         — базовое обучение (lr=0.1, epochs=100, batch=32)
├── experiments.py   — три эксперимента (lr / batch_size / init)
├── my_dataset.py    — бонус: собственный генератор датасетов
└── lab2.py          — точка входа, запускает всё
```

## Быстрый старт

```bash
pip install -r requirements.txt

# Запустить всю лабораторную
python lab2.py

# Или по частям
python data.py          # шаги 1–4: данные и визуализация
python train.py         # шаги 5–7: обучение, loss, граница
python experiments.py   # шаг 8:    три эксперимента
python my_dataset.py    # бонус 1:  собственный генератор
```

## Бонусное задание 1 — собственный генератор

Файл `my_dataset.py` реализует три генератора без sklearn:

### `make_blobs()` — гауссовые облака
```python
X, y = make_blobs(
    n_samples=500,
    center0=(-2, -2),   # центр класса 0
    center1=( 2,  2),   # центр класса 1
    cov=np.eye(2),      # ковариационная матрица
    noise=0.1,          # вероятность перевернуть метку
)
```

### `make_xor()` — паттерн XOR
```python
X, y = make_xor(
    n_samples=500,
    spread=0.4,   # разброс вокруг каждого из 4 углов
    noise=0.05,
)
```

### `make_circles()` — внутри / снаружи окружности
```python
X, y = make_circles(
    n_samples=500,
    r_inner=1.0,   # радиус внутреннего круга (класс 1)
    r_outer=2.5,   # внешний радиус кольца    (класс 0)
    noise=0.05,
)
```

### Выводы о границах применимости

| Тип данных         | Accuracy | Вывод                              |
|--------------------|----------|------------------------------------|
| Гауссовые облака   | ~99–100% | Линейно разделимы — перцептрон ✅   |
| XOR                | ~43–52%  | Хуже случайности — прямая бесполезна ❌ |
| Окружность         | ~51–53%  | Граница кривая — прямая не поможет ❌  |

Однослойный перцептрон строит линейную границу. Для нелинейных задач (XOR, окружности) необходимы либо **полиномиальные признаки**, либо **многослойные нейронные сети**.

## Технические детали

- **Функция потерь**: бинарная кросс-энтропия BCE
- **Оптимизация**: мини-батчевый SGD с перемешиванием на каждой эпохе
- **Стандартизация**: Z-score, параметры считаются только по train
- **Зависимости**: `numpy`, `matplotlib`, `scikit-learn` (только для `make_classification` и `train_test_split`)
